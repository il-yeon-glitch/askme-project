프로젝트 상태를 점검해줘. 아래 항목을 순서대로 실행하고 결과를 정리해줘:
 
1. **린트 검사**: `npx eslint client/src/ server/` 실행
   - 에러가 있으면 목록으로 정리
   - 경고는 개수만 알려줘
 
2. **서버 기동 테스트**: 서버를 시작하고 기본 API 엔드포인트에 요청을 보내 정상 응답을 확인
   - 서버가 정상 기동되면 "서버 OK"
   - 실패하면 에러 메시지 포함
 
3. **종합 판정**: 모든 항목이 통과하면 "✅ 프로젝트 상태 양호", 하나라도 실패하면 "❌ 수정 필요" + 실패 항목 목록
